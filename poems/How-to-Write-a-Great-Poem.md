# How to Write a Great Poem  

Empty the quiver, take aim,   
&nbsp;&nbsp;&nbsp;&nbsp;take every shot.  
Fletch, restock, release,   
&nbsp;&nbsp;&nbsp;&nbsp;without a second thought.  

_by Ravi Mynampaty_  
July 11, 2023
